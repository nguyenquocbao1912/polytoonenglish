# Polytoon English 🇬🇧 

A comprehensive and interactive English learning web application built to help users improve their vocabulary, practice skills, and take mock tests.

## 🚀 Key Features

*   **Vocabulary Builder:** Interactive flashcards sorted by topics for effective memorization.
*   **Skill Practice:** Targeted practice sessions divided by specific English test parts (e.g., Reading, Listening).
*   **Mock Tests Engine:** Features both "Mini Tests" for quick practice and "Full Tests" for real exam simulation.
*   **Progress Tracking:** Automatically tracks user's learning history, vocabulary mastery, and mock test scores.
*   **Authentication:** Secure user registration and login utilizing Supabase Auth.
*   **High Performance:** Optimized with Next.js App Router (SSR/SSG/ISR) and load-tested to handle concurrent users efficiently.

## 💻 Tech Stack

This project is built with modern, cutting-edge web technologies:

*   **Framework:** [Next.js (App Router)](https://nextjs.org/) & [React 19](https://react.dev/)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **Styling & UI:** [Tailwind CSS v4](https://tailwindcss.com/), [Radix UI](https://www.radix-ui.com/) & [Framer Motion](https://www.framer.com/motion/)
*   **Backend & Database:** [Supabase](https://supabase.com/) (PostgreSQL & Authentication)
*   **Load Testing:** [k6](https://k6.io/)
